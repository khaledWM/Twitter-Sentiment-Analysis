# Twitter Sentiment Analysis

## Description
This project analyzes tweets from Twitter to determine sentiments expressed in them. It utilizes Python and machine learning techniques to classify tweets as positive, negative, or neutral.

## Installation
1. Clone this repository: `git clone https://github.com/khaledWM/Twitter-Sentiment-Analysis.git`
2. Install required libraries: `pip install -r requirements.txt`

## Usage
To analyze sentiments of tweets:
1. Update `tweet_data.csv` with your Twitter data.
2. Run `python sentiment.py`
3. Check the output for sentiment analysis results.
